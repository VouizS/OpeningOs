# Opening OS

Opening OS é um launcher estilo IPTV/TV Box para projetores Android.

## v0.1.1 — Projector UI Fix

Correção baseada no primeiro teste real no projetor Android 11.

### Ajustes

- interface redesenhada para alto contraste em projeção;
- fundo preto sólido com foco âmbar;
- escala adaptativa por resolução real da tela;
- leitura e exibição de DPI/density para diagnóstico;
- redução de textos e cards sobrepostos;
- menu superior mais compacto;
- lista lateral com rolagem visual;
- aba APPS agora mostra todos os aplicativos instalados, não só os primeiros;
- adicionadas seções CLIMA e JOGOS sem dados falsos;
- botão em CONFIG para tentar abrir as configurações de app Home;
- suporte a LEANBACK_LAUNCHER para aparecer melhor em sistemas Android TV/projetor.

### Observações honestas

- Clima e jogos ao vivo ainda não usam API real. A seção existe, mas não inventa placar, tabela ou previsão.
- Se a opção de definir Opening OS como Home não aparecer, o firmware do projetor pode estar travando o launcher padrão. Isso será tratado em uma correção específica depois do próximo teste.
