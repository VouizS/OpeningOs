# Opening OS

Opening OS é um launcher estilo IPTV/TV Box para projetores Android.

## v0.1.0 — IPTV Launcher Base

Primeira base visual e funcional:

- app HOME/Launcher para substituir a tela inicial padrão;
- interface escura estilo IPTV/TV Box;
- navegação por controle remoto usando D-pad/OK;
- seções: TV, Apps, Internet, Arquivos, Ambiente e Config;
- listagem inicial de apps instalados;
- atalhos para YouTube, Chrome, configurações, arquivos e overlay;
- Modo Ambiente básico com relógio após inatividade;
- estrutura preparada para Floating Layer futuro.

## Observações

- O atalho HDMI depende do firmware do projetor e será mapeado após testes reais.
- A camada floating real será implementada em versões futuras com permissão de sobreposição.
